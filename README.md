# Botanica Project

## Project Description
This is a clone of the Botanica demo from ThemeForest, designed to showcase its features and layout.

## Features
- Responsive Design
- Clean User Interface
- Interactive Elements
- Beautiful Animations

## Technologies Used
- HTML
- SCSS
- CSS
- Bootstrap

## Project Structure
```
├── index.html
├── css
│   ├── styles.css
├── scss
│   ├── main.scss
├── js
│   ├── scripts.js
└── images
```

## Installation Instructions
1. Clone the repository:
   ```
   git clone https://github.com/iremsultanorun/botanica-project.git
   ```
2. Navigate to the project directory:
   ```
   cd botanica-project
   ```
3. Open `index.html` in your browser to view the project.

## Usage Instructions
Simply open the `index.html` file in a web browser to see the Botanica demo clone in action.
